# **App Name**: PezekaTrack

## Core Features:

- Loan Record Entry: Input and store customer loan details, including disbursement amount, due dates, and repayment schedule (daily, weekly, monthly).
- Due Date Tracker: Automatically identify and flag members with upcoming or overdue payments upon login.
- Financial Record Keeping: Maintain records of daily expenses (facilitation), payouts (customer disbursements with facilitation/transaction costs), and receipts (customer payments).
- Data Export: Generate downloadable Excel files for loan records, financial summaries, customer loan breakdowns, and statements.
- User Authentication: Implement separate login portals and permission sets for finance personnel and general staff.
- Loan Breakdown Download: Allow users to download detailed loan breakdowns and customer statements in PDF format.

## Style Guidelines:

- Primary color: Soft light blue (#ADD8E6), creating a professional and calming ambiance.
- Background color: Off-white (#F5F5F5), provides a clean and spacious backdrop.
- Accent color: Slightly darker blue (#77B5FE), for interactive elements and highlights to maintain consistency.
- Body and headline font: 'PT Sans' for readability and a modern feel.
- Use simple, clean icons for navigation and data representation.
- Maintain a clean and organized layout with clear information hierarchy.
- Subtle transitions and feedback animations for a smooth user experience.
